Architecture Canvas

This is what we recommend for your infrastructure stack.

When a user sends a message, Guardrails check it for safety and compliance. Inference & Routing sends the request to your AI model, using Knowledge Base for document retrieval and Context Cache for conversation history. MCP & Tool Use lets the model call tools (e.g. case status, deadlines). Deploy & Observe runs the app and monitors it.
